# Calculator with AVR

## Description
This is a simple calculator with AVR ATmega32 , LCD and 4x4 keypad.

## Peripherals
- AVR ATmega8
    - MCAL
        - DIO
    - HAL
        - LCD
        - Keypad
    - APP
        - Calculator    
- LCD 16x2
- 4x4 keypad


## How to use it
1. Connect your AVR to the keypad
2. Connect your AVR to the screen
3. Compile and flash the program
4. Enjoy!
